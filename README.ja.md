# ポケモン ダメージ計算　対応ゲームバージョン：Pokémon Champions M-A
## 基本画面
![基本画面](https://github.com/user-attachments/assets/b55c9a71-30a4-487e-a1ec-2b6f797314d1)

* 上部のアプリバーのボタンは順に：ダーク/ライトテーマ切り替え / 言語切り替え（端末ごとに設定を記憶）
* メイン画面でポケモンの基本数値を設定できます。新努力値システムに対応しており、最大値は32、デフォルトの合計上限は66です
* ポケモンを選択すると自動的に検索されます。該当するポケモンが見つからない場合、画像はタマゴのまま表示されます
* ***すべての言語で英語検索に対応！ポケモン・わざ・どうぐ・とくせいを入力方法を切り替えずに検索できます！***
* 一部の上位ポケモンはよく使われるわざが自動入力されます

## フィールド効果
![フィールド効果](https://github.com/user-attachments/assets/37d568f6-8383-4131-a1b3-4cf0eef07bed)
* てんき・フィールドなどの設定は右サイドパネルにあります。対応するアイコンをクリックすると有効化されます。PC版では常に表示され、スマートフォン版では右上のアイコンで開閉するメニューになります

## ダメージ計算画面
![ダメージ計算画面](https://github.com/user-attachments/assets/615f298a-30c0-4543-ba2d-9e106b72bedb)
* デフォルトではポケモン1がポケモン2を攻撃します。双方のポケモンとわざを設定するとダメージが自動計算されます。パネルは下部に折りたたむことができます
* ポケモンを入れ替えたい場合は、ポケモン選択エリア右側のスワップボタンをクリックしてください

## 保存と読み込み
![保存](https://github.com/user-attachments/assets/07db234e-ba15-49ab-892d-195efbf79648)
* ポケモン情報欄の右側のアイコンをクリックすると、設定した数値を保存・読み込みして繰り返し使用できます
* 保存時にメモを入力でき、そのポケモンの調整が対策しているポケモンなどを記録できます！

![読み込み](https://github.com/user-attachments/assets/00ee4017-370e-4e11-ad64-1db673aeb1ba)
* 読み込み時にポケモンの画像にカーソルを合わせるとメモが表示されます
* 複数のポケモンが保存されている場合、ドラッグ＆ドロップで順番を変更できます
* ***読み込みはPokepasteフォーマットのインポートに対応！！一度に最大6匹まで読み込めます！！***
