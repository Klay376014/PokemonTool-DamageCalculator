# 寶可夢傷害計算機　對應遊戲版本：Pokemon Champions M-A
## 基本介面
![基本介面](https://github.com/user-attachments/assets/b55c9a71-30a4-487e-a1ec-2b6f797314d1)

* 上方App bar的用途依序為︰深淺色主題 / 語言切換（隨裝置記憶設定）
* 在中央畫面可以設定寶可夢的基本數值，目前對應新努力值版本，最大為 32，預設總和上限為 66
* 選擇寶可夢後會進行搜尋，如果沒有該寶可夢的時候圖片會維持顯示為蛋
* ***所有語言皆支援英文搜尋！讓你搜尋寶可夢/招式/道具/特性時不用切輸入法選字！***
* 部分熱門寶可夢會自動帶入常用招式

## 場地效果
![場地效果](https://github.com/user-attachments/assets/37d568f6-8383-4131-a1b3-4cf0eef07bed)
* 天氣、場地等設定放在右側面板，點選相對應圖示即可啟用，電腦版會常駐開啟，手機版則會成為收合選單，透過右上角 icon 觸發

## 傷害計算介面
![傷害計算介面](https://github.com/user-attachments/assets/615f298a-30c0-4543-ba2d-9e106b72bedb)
* 基本預設為寶可夢1攻擊寶可夢2，在設定好雙方寶可夢及招式後即自動計算傷害，面板可收合至下方
* 如需交換寶可夢位罝則點選寶可夢選擇區塊右方的交換鈕即可交換兩邊資訊

## 儲存與讀取
![儲存](https://github.com/user-attachments/assets/07db234e-ba15-49ab-892d-195efbf79648)
* 在寶可夢資訊的地方，點選右邊icon可以進行儲存與讀取，重複使用曾經設定過的數值
* 儲存時可以輸入備註，可以註記該寶可夢調整針對的對象！

![讀取](https://github.com/user-attachments/assets/00ee4017-370e-4e11-ad64-1db673aeb1ba)
* 讀取時將游標移至寶可夢圖片上即可顥示備註
* 存有多隻寶可夢時可以進行拖拉調整順序
* ***讀取支援匯入pokepaste格式!!最多一次可以讀取6隻寶可夢!!***
